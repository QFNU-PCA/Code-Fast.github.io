			<h1>C++ 容器类 <code>&lt;queue&gt;</code></h1>&#13;
&#13;
<p>C++ 标准库中的 <code>&lt;queue&gt;</code> 头文件提供了队列（Queue）数据结构的实现。队列是一种先进先出（FIFO, First In First Out）的数据结构，它允许在一端添加元素（称为队尾），并在另一端移除元素（称为队首）。</p>&#13;
&#13;
<p>队列是一种线性数据结构，它遵循以下规则：</p>&#13;
<ul>&#13;
<li>元素只能从队尾添加。</li>&#13;
<li>元素只能从队首移除。</li>&#13;
</ul>&#13;
<h3>语法</h3>&#13;
<p>在 C++ 中，队列的语法如下：</p>&#13;
<pre>#include &lt;queue&gt;&#13;
&#13;
// 声明队列&#13;
std::queue&lt;Type&gt; q;</pre>&#13;
<p>这里 <code>Type</code> 是队列中存储元素的数据类型。</p>&#13;
<h3>常用操作</h3>&#13;
<p>队列提供了以下常用操作：</p>&#13;
<ul>&#13;
<li><code>empty()</code>: 检查队列是否为空。</li>&#13;
<li><code>size()</code>: 返回队列中的元素数量。</li>&#13;
<li><code>front()</code>: 返回队首元素的引用。</li>&#13;
<li><code>back()</code>: 返回队尾元素的引用。</li>&#13;
<li><code>push()</code>: 在队尾添加一个元素。</li>&#13;
<li><code>pop()</code>: 移除队首元素。</li>&#13;
</ul>&#13;

<h3>注意事项</h3>&#13;
<ul>&#13;
<li>队列不允许随机访问元素，即不能直接通过索引访问队列中的元素。</li>&#13;
<li>队列的实现通常使用链表或动态数组，这取决于具体的实现。</li>&#13;
</ul>&#13;
