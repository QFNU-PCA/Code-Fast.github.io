			<h1>C <span class="color_h1">基础语法</span></h1>&#13;
&#13;
&#13;
<p>C 语言是一种通用的编程语言，广泛应用于系统编程、嵌入式开发和高性能计算等领域。</p><p>C 语言具有高效、灵活、可移植性强等特点，是许多其他编程语言的基础。</p>&#13;
<p>在 C 语言中，令牌（Token）是程序的基本组成单位，编译器通过对源代码进行词法分析，将代码分解成一个个的令牌。</p><p>C 语言的令牌主要包括以下几种类型：</p>&#13;
<ul><li><strong>关键字（Keywords）</strong></li><li><strong>标识符（Identifiers）</strong></li><li><strong>常量（Constants）</strong></li><li><strong>字符串字面量（String Literals）</strong></li><li><strong>运算符（Operators）</strong></li><li><strong>分隔符（Separators）</strong></li></ul>&#13;
<h2 class="tutheader">C 程序的基本结构</h2><p>&#13;
这是一个简单的 C 语言程序，可以输出 "Hello, World!"：</p>&#13;
<div class="example"><h2 class="example">实例</h2> <div class="example_code">
<span style="color: #085;">#include &lt;stdio.h&gt;</span><br/>
<br/>
<span style="color: #993333;">int</span> main<span style="color: #000;">(</span><span style="color: #000;">)</span> <span style="color: #000;">{</span><br/>
    <span style="color: #000066;">printf</span><span style="color: #000;">(</span><span style="color: #a11;">"Hello, World!<span style="color: #000099; font-weight: bold;">\n</span>"</span><span style="color: #000;">)</span><span style="color: #339933;">;</span><br/>
    <span style="color: #708;">return</span> <span style="color: #164;">0</span><span style="color: #339933;">;</span><br/>
<span style="color: #000;">}</span><br/>
</div></div>&#13;
<p>以上代码组成结构如下：</p>&#13;
<ul><li><strong>预处理指令</strong>：如 <code>#include</code> 和 <code>#define</code>。</li><li><strong>主函数</strong>：每个 C 程序都至少有一个 <code>main()</code> 函数。</li><li><strong>变量声明</strong>：声明程序中使用的变量。</li><li><strong>函数定义</strong>：定义程序中使用的函数。</li></ul>&#13;
&#13;
&#13;
<h2 class="tutheader">分隔符</h2>&#13;
<p>分隔符用于分隔语句和表达式，常见的分隔符包括：</p>&#13;
<ul><li><strong>逗号（,）</strong>：用于分隔变量声明或函数参数。</li><li><strong>分号（;）</strong>：用于结束语句。</li><li><strong>括号</strong>：<ul><li>圆括号（<code>()</code>）用于分组表达式、函数调用。</li><li>花括号（<code>{}</code>）用于定义代码块。</li><li>方括号（<code>[]</code>）用于数组下标。</li></ul></li></ul>&#13;
<p>在 C 程序中，分号 <span class="marked">;</span> 是语句结束符，也就是说，每个语句必须以分号结束，它表明一个逻辑实体的结束。</p>&#13;
<p>例如，下面是两个不同的语句：</p>&#13;
<pre>&#13;
printf("Hello, World! \n");&#13;
return 0;&#13;
</pre>&#13;
<p>个单独的分号也可以作为一个空语句，表示什么都不做。例如：</p>&#13;
<pre>&#13;
;</pre>&#13;
<h2 class="tutheader">注释</h2>&#13;
&#13;
<p/><p><strong>C 语言有两种注释方式：</strong></p>&#13;
<pre>&#13;
// 单行注释&#13;
</pre>&#13;
&#13;
<p>以 <span class="marked">//</span> 开始的单行注释，这种注释可以单独占一行。</p>&#13;
<pre>&#13;
/* 单行注释 */&#13;
/* &#13;
 多行注释&#13;
 多行注释&#13;
 多行注释&#13;
 */&#13;
</pre>&#13;
<p> <span class="marked">/* */ </span> 这种格式的注释可以单行或多行。</p>&#13;
<div class="example"><h2 class="example">实例</h2> <div class="example_code">
<span style="color: #666666; font-style: italic;">// 这是单行注释</span><br/>
<br/>
<span style="color: #808080; font-style: italic;">/*<br/>
这是多行注释<br/>
它可以跨越多行<br/>
*/</span><br/>
<br/>
<span style="color: #993333;">int</span> main<span style="color: #000;">(</span><span style="color: #000;">)</span> <span style="color: #000;">{</span><br/>
    <span style="color: #666666; font-style: italic;">// 打印一条消息</span><br/>
    <span style="color: #000066;">printf</span><span style="color: #000;">(</span><span style="color: #a11;">"Hello, World!<span style="color: #000099; font-weight: bold;">\n</span>"</span><span style="color: #000;">)</span><span style="color: #339933;">;</span><br/>
    <span style="color: #708;">return</span> <span style="color: #164;">0</span><span style="color: #339933;">;</span><br/>
<span style="color: #000;">}</span><br/>
</div></div>&#13;
&#13;
<p>您不能在注释内嵌套注释，注释也不能出现在字符串或字符值中。</p>&#13;
&#13;
<h2 class="tutheader">标识符</h2>&#13;
<p>标识符是程序中变量、函数、数组等的名字。标识符由字母（大写或小写）、数字和下划线组成，但第一个字符必须是字母或下划线，不能是数字。</p>&#13;
<p>一个标识符以字母 A-Z 或 a-z 或下划线 _ 开始，后跟零个或多个字母、下划线和数字（0-9）。&#13;
</p>&#13;
<p>C 标识符内不允许出现标点字符，比如 @、$ 和 %。C 是<b>区分大小写</b>的编程语言。因此，在 C 中，<i>Manpower</i> 和 <i>manpower</i> 是两个不同的标识符。下面列出几个有效的标识符：</p>&#13;
<pre>&#13;
mohd       zara    abc   move_name  a_123&#13;
myname50   _temp   j     a23b9      retVal&#13;
</pre>&#13;
&#13;
<h2 class="tutheader">常量</h2><p>&#13;
常量是固定值，在程序执行期间不会改变。</p><p>常量可以是整型常量、浮点型常量、字符常量、枚举常量等。</p>&#13;
<pre>&#13;
const int MAX = 100;  // 整型常量&#13;
const float PI = 3.14;  // 浮点型常量&#13;
const char NEWLINE = '\n';  // 字符常量</pre>&#13;
<h2 class="tutheader">字符串字面量</h2><p>&#13;
字符串字面量是由双引号括起来的字符序列。</p><p>字符串末尾会自动添加一个空字符 <span class="marked">\0</span>。</p>&#13;
<pre>&#13;
char greeting[] = "Hello, World!";</pre>&#13;
<h2 class="tutheader">运算符（Operators）</h2><p>&#13;
运算符用于执行各种操作，如算术运算、逻辑运算、比较运算等。</p><p>C 语言中的运算符种类繁多，常见的包括：</p>&#13;
<ul><li><strong>算术运算符</strong>：<code>+</code>, <code>-</code>, <code>*</code>, <code>/</code>, <code>%</code></li><li><strong>关系运算符</strong>：<code>==</code>, <code>!=</code>, <code>&gt;</code>, <code>&lt;</code>, <code>&gt;=</code>, <code>&lt;=</code></li><li><strong>逻辑运算符</strong>：<code>&amp;&amp;</code>, <code>||</code>, <code>!</code></li><li><strong>位运算符</strong>：<code>&amp;</code>, <code>|</code>, <code>^</code>, <code>~</code>, <code>&lt;&lt;</code>, <code>&gt;&gt;</code></li><li><strong>赋值运算符</strong>：<code>=</code>, <code>+=</code>, <code>-=</code>, <code>*=</code>, <code>/=</code>, <code>%=</code></li><li><strong>其他运算符</strong>：<code>sizeof</code>, <code>?:</code>, <code>&amp;</code>, <code>*</code>, <code>-&gt;</code>, <code>.</code></li></ul>&#13;
<pre>int a = 5, b = 10;&#13;
int sum = a + b;  // 使用算术运算符 +&#13;
int isEqual = (a == b);  // 使用关系运算符 ==</pre>&#13;

<h2 class="tutheader">C 中的空格</h2>&#13;
<p>只包含空格的行，被称为空白行，可能带有注释，C 编译器会完全忽略它。</p>&#13;
<p>在 C 中，空格用于描述空白符、制表符、换行符和注释。空格分隔语句的各个部分，让编译器能识别语句中的某个元素（比如 int）在哪里结束，下一个元素在哪里开始。因此，在下面的语句中：</p>&#13;
<pre>&#13;
int age;&#13;
</pre>&#13;
<p>在这里，int 和 age 之间必须至少有一个空格字符（通常是一个空白符），这样编译器才能够区分它们。另一方面，在下面的语句中：</p>&#13;
<pre>&#13;
fruit = apples + oranges;   // 获取水果的总数&#13;
</pre>&#13;
<p>fruit 和 =，或者 = 和 apples 之间的空格字符不是必需的，但是为了增强可读性，您可以根据需要适当增加一些空格。</p>			<!-- 其他扩展 -->
