			<h1>Python statistics 模块</h1>&#13;
<p>Python statistics 是标准库中的一个模块，模块提供了许多基本统计计算的函数。</p>&#13;
&#13;
<p>statistics 模块是在 Python 3.4 版本中新增加的，可以帮助我们分析和计算数据集的统计特征。</p>&#13;
&#13;
<p>&#13;
要使用 statistics 函数必须先导入：</p>&#13;
&#13;
<pre>import statistics</pre>&#13;
